This project is built to detect fake job postings using advanced Machine Learning models like SVM, Extra Trees, Passive Aggressive, and XGBoost. It analyzes different features such as job descriptions, company profiles, requirements, and metadata to identify suspicious or fraudulent job postings. The goal is to protect job seekers from scams, spam, and misleading advertisements that can result in wasted time or even financial loss. By leveraging both structured attributes and text-based data, the system achieves a balance between accuracy and efficiency. The interactive dashboard allows users to visualize model performance, compare training times, and evaluate metrics like Accuracy and F1 Score. This application demonstrates the practical use of data science to solve real-world problems in employment security and trust.
